后台运行

后台运行命令

将前台命令放到挂起（stopped）
Ctrl + Z 查看后台工作项目

操作后台工作项目

  • 终止后台工作项目 kill jobNumber
  • 调至前台继续运行 fg jobNumber
  • 在后台执行挂起的工作项目 bg jobNumber

    vi查找

    在命令模式下敲斜杆( / )这时在状态栏（也就是屏幕左下脚）就出现了 “/” 然后输入你要查找的关键字敲回车就可以了。
    如果你要继续查找此关键字，敲字符 n 就可以继续查找了。 敲字符N（大写N）就会向前查询；
