缩进

Python没有分割代码块符号，以缩进以及冒号:区分层次，一般以4个空格为基本缩进单位。

命名规范

  • 模块名：全部小写 + 下划线_
  • 包名：全部小写，不推荐使用下划线
  • 类名：Pascal风格（首字母大写）
  • 模块内部类型：_ + Pascal风格
    使用单下划线_开头的变量和函数是受保护的，在使用from xxx import * 时不会被导入；使用双下划线__开头的变量或方法是类私有的
  • 函数、类的属性和方法：同模块命名
  • 常量：全部大写
