Why React Native

React Native е’ҢеүҚз«ҜжҠҖжңҜз”ҹжҖҒйҮҚеҗҲеәҰеҫҲй«ҳпјҢеӯҰд№ жҲҗжң¬дҪҺ, дёҖд»Ҫжәҗз ҒеҸҜд»ҘеҗҢж—¶зј–иҜ‘жҲҗ Android е’Ң iOS еҺҹз”ҹеә”з”Ё. йҖӮеҗҲжӣҙж–°иҝӯд»Јиҫғеҝ«зҡ„йЎ№зӣ®
еҗҢж—¶ е…¶еә•еұӮжһ¶жһ„еёҲж №жӨҚдәҺз§»еҠЁз«ҜеҺҹз”ҹAPI еҸҜд»ҘиҮӘе®ҡд№үеҺҹз”ҹжЁЎеқ— д»ҘеҸҠдјҳеҢ–жҖ§иғҪ

жң¬иҙЁпјҲnatureпјү

еңЁ Android ејҖеҸ‘дёӯжҳҜдҪҝз”Ё Kotlin жҲ– Java жқҘзј–еҶҷи§ҶеӣҫпјӣеңЁ iOS ејҖеҸ‘дёӯжҳҜдҪҝз”Ё Swift жҲ– Objective-C жқҘзј–еҶҷи§ҶеӣҫгҖӮеңЁ React Native дёӯпјҢеҲҷдҪҝз”Ё React з»„д»¶йҖҡиҝҮ JavaScript жқҘи°ғз”Ёиҝҷдәӣи§ҶеӣҫгҖӮеңЁиҝҗиЎҢж—¶пјҢReact Native дёәиҝҷдәӣз»„д»¶еҲӣе»әзӣёеә”зҡ„ Android е’Ң iOS и§ҶеӣҫгҖӮз”ұдәҺ React Native з»„д»¶е°ұжҳҜеҜ№еҺҹз”ҹи§Ҷеӣҫзҡ„е°ҒиЈ…пјҢеӣ жӯӨдҪҝз”Ё React Native зј–еҶҷзҡ„еә”з”ЁеӨ–и§ӮгҖҒж„ҹи§үе’ҢжҖ§иғҪдёҺе…¶д»–д»»дҪ•еҺҹз”ҹеә”з”ЁдёҖж ·гҖӮжҲ‘д»¬е°Ҷиҝҷдәӣе№іеҸ°ж”ҜжҢҒзҡ„з»„д»¶з§°дёәеҺҹз”ҹз»„д»¶гҖӮ

еңЁиҫғж—©зҡ„React NativeдёӯйҖҡиҝҮз§°дёәBridgeзҡ„ејӮжӯҘжңәеҲ¶е®ҢжҲҗjsдёҺеҺҹз”ҹд»Јз Ғд№Ӣй—ҙзҡ„йҖҡдҝЎпјҢ0.74зүҲжң¬ејҖе§Ӣй»ҳи®ӨдҪҝз”ЁBridgeless modeпјҢи°ғз”ЁJSIи®ҝй—®еҺҹз”ҹд»Јз Ғд»ҺиҖҢжҸҗй«ҳжҖ§иғҪе’Ңе“Қеә”йҖҹеәҰ

React Native е…Ғи®ёжӮЁдёә Android е’Ң iOS жһ„е»әиҮӘе·ұзҡ„ Native ComponentsпјҲеҺҹз”ҹз»„д»¶пјүпјҢд»Ҙж»Ўи¶іжӮЁејҖеҸ‘еә”з”ЁзЁӢеәҸзҡ„зӢ¬зү№йңҖжұӮгҖӮReact Native иҝҳеҢ…жӢ¬дёҖз»„еҹәжң¬зҡ„пјҢйҡҸж—¶еҸҜз”Ёзҡ„еҺҹз”ҹз»„д»¶пјҢжӮЁеҸҜд»ҘдҪҝз”Ёе®ғд»¬жқҘжһ„е»әжӮЁзҡ„еә”з”ЁзЁӢеәҸгҖӮиҝҷдәӣжҳҜ React Native зҡ„ж ёеҝғз»„д»¶гҖӮ
