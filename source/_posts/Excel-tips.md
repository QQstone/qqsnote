#### 随机数
```
=rand() //生成随机数区间[0,1]
=randbetween(100,999)//生成范围内随机整数
```
#### 随机字符
```
=CHAR(INT(RAND()*25+65))
```
#### 字符串拼接
```
=concat("id","0001")
```
#### 随机日期
```
=TEXT(RAND()*("2019-11-11"-"2018-11-11")+"2018-11-11","e/m/d")
```
